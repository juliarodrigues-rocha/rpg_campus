## Sobre o projeto

Primeira tela funcional do jogo RPG Campus desenvolvida com Flutter e Dart.

## Estrutura de pastas

```
lib/
  main.dart         # Ponto de entrada do app
  models/           # Modelos de dados
  data/             # Dados estáticos / repositórios locais
  services/         # Serviços e lógica de negócio
  screens/          # Telas do app
    home_screen.dart
  widgets/          # Widgets reutilizáveis
```

## Como rodar

```bash
flutter pub get
flutter run
```
